# PM Vaporwave Tarot - Asset Collection Guide

## Project Vision
Retro computing (80s/90s) + Classical Roman/Greek statues + Vaporwave aesthetic + Product Management wisdom

## Asset Categories

### 1. STATUES (Angels, Roman/Greek Figures)

**What we need:**
- Angelic figures (like Bernini angels)
- Classical poses (pointing, sitting, standing)
- Dramatic gestures for each arcana
- Columns and architectural elements

**Where to find:**
- Cults3D: "Complete Pack Tarot Rider Figures" ($19.95) - PURCHASED/PLANNED
- Sketchfab: Search "roman statue" (filter: Downloadable)
- Met Museum Open Access: High-res classical sculpture photos
- Unsplash: "roman statue", "angel statue", "marble sculpture"
- Wikimedia Commons: "Bernini angel", "Vatican sculpture"

**File format needed:**
- Photos: High-res JPG (2000px+)
- 3D models: OBJ, FBX (for rendering in Blender)
- Cutouts: PNG with transparent background

### 2. OLD TECH (CRT, VHS, Computers, Phones)

**Specific items needed:**
- CRT monitors (various angles)
- VHS tapes
- Cassette tapes
- Floppy disks (3.5" and 5.25")
- Old desktop computers (IBM PC, early Mac)
- Retro phones
- Keyboards (mechanical, vintage)
- Computer mice with cords
- Server racks

**Where to find:**
- Unsplash: "vintage computer", "CRT monitor", "retro technology"
- Pexels: "old computer", "VHS tape", "cassette tape"
- Internet Archive: Computer catalogs and ads (public domain)
- PNGWing: Pre-cut PNG tech items

### 3. JAPANESE ELEMENTS (Kanji, Text)

**Kanji for PM concepts:**
- プロダクト (purodakuto) - Product
- ユーザー (yūzā) - User
- スプリント (supurinto) - Sprint
- データ (dēta) - Data
- チーム (chīmu) - Team
- 目標 (mokuhyō) - Goal
- 成長 (seichō) - Growth
- 改善 (kaizen) - Improvement
- 価値 (kachi) - Value
- 計画 (keikaku) - Plan

**Fonts to download:**
- Noto Sans JP (Google Fonts)
- Sawarabi Gothic (Google Fonts)

### 4. NATURE (Sun, Palm Trees, Mountains, Desert)

**Specific needs:**
- Sunset/sunrise gradients (pink/purple/cyan)
- Palm tree silhouettes
- Desert landscapes
- Mountain ranges
- Grid floors with perspective (Tron-style)
- Starfields/cosmic backgrounds

**Where to find:**
- Unsplash: "palm tree sunset", "desert landscape", "mountain sunrise"
- Search "vaporwave sunset" for pre-styled images

### 5. FLOWERS

**Types needed:**
- Roses (various colors)
- Lilies
- Generic botanical elements
- Can be stylized/vectorized

**Where to find:**
- Unsplash: "rose", "lily", "flower"
- Vecteezy: Free flower vectors
- Freepik: Flower illustrations

### 6. EFFECTS & TEXTURES

**Essential overlays:**
- Scanlines (horizontal lines for CRT effect)
- RGB glitch overlays
- Chromatic aberration textures
- Grid patterns (perspective and flat)
- Light leaks
- VHS artifacts
- Noise/grain textures

**Where to find:**
- Search "scanline overlay png"
- Search "CRT overlay texture"
- Lost & Taken: Free texture overlays
- Create custom in Photoshop/Procreate

**Photoshop actions to find:**
- "Vaporwave effect action"
- "Glitch effect action"
- "Duotone effect action"

## Color Palette Reference

**Primary colors:**
- Neon Cyan: #00FFFF
- Hot Magenta: #FF00FF
- Electric Pink: #FF10F0
- Neon Green: #39FF14

**Background:**
- Pure Black: #000000

**Gradients:**
- Pink → Purple → Cyan (classic vaporwave)
- Orange → Pink (sunset)
- Blue → Purple (mystical night)

## Asset Requirements for Print

**Technical specs:**
- Resolution: 300 DPI minimum
- Color mode: CMYK (for print)
- File format: High-res JPG or PNG
- Final card size: 2.75" × 4.75" = 825px × 1425px at 300 DPI

**Quality checklist:**
- [ ] Image is sharp, not blurry
- [ ] High enough resolution to scale
- [ ] Clean background or easy to mask
- [ ] Good lighting/contrast
- [ ] Matches vaporwave aesthetic potential
